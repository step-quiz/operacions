# Step Quiz: Matemàtiques Interactives

Aquest repositori conté un conjunt de jocs i activitats interactives en format web (HTML/JS/CSS) dissenyades per ajudar els estudiants a practicar mecànicament diferents operacions matemàtiques de nivell d'ESO.

## ⚙️ Com funciona?
No requereix instal·lació ni servidor (Backend). Són arxius totalment estàtics. 
Pots provar-ho simplement obrint `index.html` amb qualsevol navegador web. 

A través de l'arxiu `index.html` es poden generar enllaços personalitzats per als alumnes, configurant els paràmetres següents mitjançant la URL:
* Nombre de sessions (`totalsessions`)
* Operacions per sessió (`totaloperations`)
* Intents màxims (`maxintents`)

## 🛠️ Tecnologies
* HTML5 / CSS3 purs (sense frameworks).
* Vanilla JavaScript (ES6).
